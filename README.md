# VisionGen

VisionGen is an AI-powered image generation tool built on top of the Google Opal framework. 

## 🚀 What is VisionGen?

VisionGen allows you to seamlessly generate high-quality images directly from your text prompts. By leveraging Opal's advanced multimodal capabilities, it transforms your descriptive text into detailed visual art. 

## 📸 What can you do with it?

* **Enter Text Prompts:** Input any descriptive text to instantly generate an image.
* **Save and Download:** Automatically download and save your generated images directly to your local device.
